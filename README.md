# 所用技术

- 语言： Kotlin
- 框架： SpringBoot
- 消息队列: RabbitMQ
- 构建工具：Gradle

# 主要功能

- [x] 登录 (学号)   
- [ ] 权限认证  
- [x] 后台异常邮件发送
- [x] 试题获取
- [x] 当前考试试题
- [x] 提交考试
- [x] 试卷评分
- [ ] 试题查看
